# Hola Webpack

- webpack, webpack-cli
- clean-webpack-plugin
- html-webpack-plugin
- css-loader, style-loader
- mini-css-extract-plugin
- babel-core, babel-loader, babel-preset-env
- node-sass, sass-loader
- ts-loader

## Project setup
```
npm install
```

## Dev mode
```
npm run dev
```

## Prod mode

npm run build
